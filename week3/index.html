Chapter 6 The Document Object Model or the DOM.

The DOM is a programming interface for HTML and XML documents that allow a structured representation of the document and how they can access it to change its structure, content, and style.

We can access the DOM of a document through the console using commands that interact with the elements of the web page and thus see the results.

Chapter 7 Events.

Events are part of the DOM and are the link between the web page and user interactions. These occur when the user interacts with the web page, there are many kinds of events and we can detect these events and invoke a function for when this occurs, there are many ways to detect an event and program how it will respond to such an event.
